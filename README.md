# XScroll 3.0

#### 主要分核心、手势、插件、应用四部分。

* 核心：xscroll.core，主要产出一个稳定的scroll容器。

* 手势：单独拆分XGesture项目去做，完全无限逼近IOS Native去做。

* 插件：更完整的插件机制，将无尽滚动(infinite)、分页器(pagination) 、侧滑编辑(swipe-editable)等作为插件处理。

* 应用：基于xscroll进行高级封装，使得更方便的进行业务场景使用。
